# Closed-Loop Cost Brain

> **Autonomous real-time AI optimization with intelligent interventions**

The Closed-Loop Cost Brain is Cost Katana's autonomous AI system that performs real-time, in-flight optimization of AI requests before they reach providers.

## Overview

The Cost Brain intercepts and optimizes every AI request with:

- **Real-time Interventions** - Model downgrades, provider switching, prompt compression
- **Budget Protection** - Automatic request blocking when budgets are exhausted
- **Active Flow Management** - Resource allocation and conflict detection
- **Learning Feedback Loops** - Continuously improves from outcomes
- **Shadow Mode** - Test optimizations without applying them

## Features

- **Autonomous Optimization** - Hands-free cost reduction
- **Intelligent Routing** - Automatically selects optimal providers
- **Budget Enforcement** - Hard limits to prevent overspending
- **Quality Preservation** - Maintains output quality while reducing costs
- **Transparency** - Full visibility into all interventions

## Quick Start

### TypeScript/Node.js
```bash
npm install cost-katana
export COST_KATANA_API_KEY=your_key_here
npx ts-node npm-package/basic-brain.ts
```

### Python
```bash
pip install costkatana
export COST_KATANA_API_KEY=your_key_here
python python-sdk/basic_brain.py
```

### HTTP REST
1. Open `http-headers/brain.http` in VS Code
2. Install [REST Client extension](https://marketplace.visualstudio.com/items?itemName=humao.rest-client)
3. Update the API key
4. Click "Send Request"
