# Cost Katana Projects Management

Organize AI usage by projects with tracking and analytics.

## Quick Start
```bash
curl -X POST https://cost-katana-backend.store/api/projects \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"name": "My Project"}'
```

## Features
✅ Project-based organization
✅ Cost tracking per project
✅ Team collaboration
✅ Analytics & reporting
